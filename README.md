# TSM Advisory Services Website

A modern, responsive website for TSM Advisory Services - a professional accounting and advisory firm.

## Features

- **Modern Design**: Clean, professional design with gradient backgrounds and smooth animations
- **Responsive Layout**: Fully responsive design that works on all devices
- **Interactive Elements**: Smooth scrolling, hover effects, and animated components
- **Contact Integration**: 
  - Email functionality (opens default email client)
  - WhatsApp integration with pre-filled messages
  - Phone number click-to-call functionality
- **Service Showcase**: Comprehensive display of all 9 service categories
- **User Experience**: Psychological engagement through modern UI/UX principles

## Services Highlighted

1. Audit & Assurance Services
2. Tax Services
3. Advisory Services
4. Risk Management & Governance
5. Compliance & Regulatory Services
6. Transaction Services
7. Financial Reporting & Accounting
8. Business Process Outsourcing
9. Capacity Building

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with flexbox/grid, animations, and responsive design
- **JavaScript**: Interactive functionality, form validation, and smooth user experience
- **Font Awesome**: Professional icons
- **Google Fonts**: Inter font family for modern typography

## Setup Instructions

1. Clone this repository to your local machine
2. Update the contact information in `index.html`:
   - Replace `[Your Address Here]` with actual address
   - Replace `[Your Phone Number]` with actual phone number
   - Update WhatsApp link with correct phone number
   - Verify email address is correct

3. Customize the content:
   - Update company description if needed
   - Modify services details as required
   - Add company logo if available
   - Update social media links

4. Deploy to GitHub Pages:
   - Push to your GitHub repository
   - Enable GitHub Pages in repository settings
   - Select source as main branch

## Contact Information to Update

Please update the following placeholders in the HTML file:

- **Address**: `[Your Address Here]`
- **Phone**: `[Your Phone Number]`
- **WhatsApp**: Update the phone number in WhatsApp links
- **Email**: Verify `info@tsmadvisory.com` is correct
